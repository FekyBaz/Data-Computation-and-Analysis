Filter-Based Feature Selection for Multi-Class Classification with LDA

1. Introduction

This project studies dimensionality reduction through filter-based feature selection and evaluates how different ranking criteria affect multi-class classification performance. The workflow applies five filter criteria (Pearson, Spearman, Chi-Square, Gini Index, and Information Gain), ranks features with each method, and trains a multi-class LDA classifier using top-k subsets. The report focuses on performance, ranking stability, and sensitivity to discretization.

2. Dataset Description

The experiments use the Wine dataset (178 samples, 13 numerical features, 3 classes). The dataset satisfies all assignment constraints: multi-class problem (3 classes), more than 6 features (13), and complete observations without missing values.

  • Samples: 178
  • Features: 13 continuous attributes
  • Classes: 3 target categories

3. Data Validation and Exploratory Analysis

Before modeling, the pipeline validates data quality and consistency by checking dataset structure, data types, missing values, duplicate rows, and class distribution. No missing values and no duplicates are found.

Exploratory analysis includes class-count visualization, per-feature distributions, class-conditional boxplots, and a correlation heatmap. These plots confirm that several features are class-informative while some pairs are strongly correlated.

Figures used: class distribution bar chart, feature histograms, class-wise boxplots, and feature correlation heatmap.

4. Feature Selection Methods

All feature selection criteria are implemented from scratch and used to produce descending feature rankings.

4.1 Pearson Correlation

For each feature \(X_j\), the absolute Pearson correlation with target \(y\) is used.

\( r_j = \left|\frac{\mathrm{cov}(X_j, y)}{\sigma_{X_j}\sigma_y}\right| \)

4.2 Spearman Rank Correlation

Feature and target values are converted to ranks, then Pearson is computed on ranked values.

\( \rho_j = \left|\mathrm{corr}(\mathrm{rank}(X_j),\mathrm{rank}(y))\right| \)

4.3 Chi-Square Statistic

Continuous features are discretized (quantile bins), then a contingency table against class labels is built. The feature score is the Chi-Square statistic:

\( \chi^2_j = \sum_{a,b} \frac{(O_{ab}-E_{ab})^2}{E_{ab}} \)

4.4 Gini Impurity Reduction

For each discretized feature, impurity reduction from parent to child partitions is computed:

\( \mathrm{GiniGain}(X_j)=\mathrm{Gini}(y)-\sum_t p_t\,\mathrm{Gini}(y\mid X_j=t) \)

4.5 Information Gain

Information Gain is computed as entropy reduction after partitioning by a discretized feature:

\( \mathrm{IG}(X_j)=H(y)-\sum_t p_t\,H(y\mid X_j=t) \)

5. Multi-Class LDA Model

Classification is performed with a manual multi-class LDA implementation. For each class, the class mean is estimated and a pooled covariance matrix is computed from training data. Prediction uses linear discriminant scores with shared covariance assumption.

\( \delta_k(x)=x^T\Sigma^{-1}\mu_k-\frac{1}{2}\mu_k^T\Sigma^{-1}\mu_k+\log\pi_k \)

The assigned class is \(\arg\max_k \delta_k(x)\). Features are standardized inside each fold before LDA training.

6. Experimental Setup

Evaluation uses 5-fold stratified cross-validation with leakage-safe design: feature ranking is recomputed using only each training fold, then top-k features are selected. The tested feature counts are:

\(k = [2, 4, 6, 8, 10, 13]\)

Performance metrics include accuracy, macro precision, macro recall, macro F1-score, and confusion matrix.

7. Results and Performance Comparison

Each method is evaluated across all k values. As k increases, performance generally improves and then saturates near the full feature set. Differences across methods are clearer at smaller k values, where ranking choice has higher impact.

Figures used: Accuracy vs Top-k Features, Macro-F1 vs Top-k Features, and confusion matrix for the best method-k setting.

8. Stability Analysis

Stability is analyzed in two ways:

  • Jaccard similarity between top-k feature sets across folds (set-level stability).
  • Spearman rank correlation between full ranking lists across methods/folds (order-level stability).

Jaccard values usually increase with larger k due to higher overlap, while rank correlations highlight how similarly methods order all features.

9. Sensitivity Analysis

Since Chi-Square, Information Gain, and Gini depend on discretization, a sensitivity study is performed with different quantile bin counts:

\(\text{bins} = [3, 5, 7, 10]\)

The analysis compares metric trends under different binning granularity and confirms that broad conclusions remain stable, although small differences can appear for specific methods.

10. Discussion of Theoretical Differences

Pearson and Spearman measure association with target labels (linear vs monotonic after ranking). They are simple and fast, but can miss class-separation patterns that are not well represented by correlation. Chi-Square, Information Gain, and Gini are class-distribution based criteria on discretized features, often capturing class purity effects more directly, but they depend on discretization quality and bin selection.

Therefore, no single criterion is universally best; method choice depends on the expected relationship type, robustness needs, and interpretability goals.

11. Conclusion

The final pipeline provides a complete filter-based dimensionality reduction study for multi-class classification. It includes dataset validation, from-scratch feature scoring, top-k experiments, manual multi-class LDA evaluation, ranking stability analysis, and discretization sensitivity testing. The results show that feature selection method, subset size, and discretization strategy jointly influence both predictive performance and ranking robustness.
